Nous vous demandons d'installer la bibliothèque Python matplotlib, car nous utiliserons sa sous-bibliothèque matplotlib.pyplot. Vous pouvez trouver des instructions ici

Nous vous demandons aussi d'installer les bibliothèques Python numpy et scipy (instructions ici), nous les utiliserons au début du second semestre.

En résumé :

  • Pour les utilisateurs de Linux, l'installation de toutes ces bibliothèques se fait très simplement, par le gestionnaire de paquets de votre choix, ou dans la console avec la commande suivante. 
     sudo apt-get install python-matplotlib python-numpy python-scipy
  • Pour les utilisateurs de Windows (et Mac), l'utilisation d'une distribution Linux adaptée est la plus simple. Nous vous conseillons Anaconda.

Cette année, les travaux pratiques en informatique se feront uniquement sous le système d'exploitation Linux (sous lequel travaillent d'ailleurs vos enseignants). Plusieurs raisons à celà.

  • C'est un système d'exploitation libre et gratuit, sous lequel l'usage des logiciels (libres) tels que Python est parfaitement adapté.
  • D'expérience, l'installation et l'usage de certains modules de Python est compliqué, voire impossible en pratique, sous Windows.
  • Pédagogiquement, manipuler Linux permet d'appréhender beaucoup de notions informatiques plus directement que sous Windows, notamment.
Pour celà, nous vous recommandons très fortement de pouvoir travailler sous Linux sur votre ordinateur personnel. Il existe plusieurs distributions de Linux, la plus connue et grand public est Ubuntu, sa version Xubuntu est légère et très pratique. Une simple recherche sur votre moteur de recherche préféré vous permettra de la télécharger. Ensuite, si Linux n'est pas votre système d'exploitation habituel, vous avez plusieurs solutions à votre disposition.
  • Installer Linux en dual boot, c'est-à-dire en parallèle de votre autre système habituel avec un choix à faire à chaque démarrage. C'est malheureusement impossible sous Mac, et presque impossible depuis Windows 8. C'est la solution la plus pratique (le système Linux est pérenne), mais un peu technique. Il faut notamment repartitionner le disque dur : il est impératif de sauvarder ses données avant de se lancer dedans.
  • Installer Linux sur un clef USB bootable. Pour cela, une distribution légère est recommandée, comme Xubuntu. Il faut alors indiquer au BIOS de l'ordinateur de démarrer en priorité à partir d'un clef USB (par rapport au disque dur habituel). À partir de Windows 8, la nouvelle gestion du BIOS rend cette méthode compliquée.
  • Travailler sous machine virtuelle. C'est la méthode la plus simple à mettre en oeuvre et qui devrait fonctionner dans tous les cas. Dans votre système d'exploitation habituel, il suffit d'installer un logicielle comme VirtualBox (gratuit). Ce dernier permet d'émuler un autre système d'exploitation, dans lequel vous pourrez travailler de manière transparente, bien qu'un peu ralentie parfois. Il suffit pour cela de réserver suffisamment d'espace disque lors de l'installation du logiciel.
Nous sommes à votre disposition pour vous aider à mettre en oeuvre une de ces trois méthodes.
